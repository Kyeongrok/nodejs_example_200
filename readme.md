## Webstorm 단축키
<pre>
Alt + 1 : 프로젝트 explorer로 이동
ESC : 소스코등 창으로 이동 
Ctrl + K : Commit
Ctrl + Shift + K : Push
Ctrl + Alt + S : 환경설정
Ctrl + T : Git pull
Ctrl + E : 최근 파일 목록 
Ctrl + Shift + f10 : 실행
Ctrl + Tab : 이전 코드 
Ctrl + D : 한줄 코드 복사 
Ctrl +  Y : 한줄 코드 삭제 
Ctrl + Shift + / : 지정 된 코드 주석
Ctrl + ALT + I : 소스정리 
</pre>
